Overview
The Elayne administration Tool is a Tool aimed to L2J Server owners that will allow them to control their server remotely and without the need to be In Game. It is based on L2J (vanilla), a Lineage2 Server emulator based on pure Java for best compatibility. The features of this tool will increase in time.
